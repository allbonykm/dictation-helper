# 💖 율이를 위한 맞춤형 받아쓰기 도우미 (PWA) 기획서

## 1. 프로젝트 개요
세상에 하나뿐인 우리 율이를 위해 아빠가 직접 만드는 **개인 맞춤형 받아쓰기 연습 앱**입니다. PWA 기술을 활용하여 스마트폰이나 태블릿 홈 화면에 설치하고, 언제 어디서든 간편하게 학습할 수 있도록 합니다.

## 2. 핵심 목표
- **율이 맞춤형**: 아이의 학습 속도와 취약점에 맞춘 개인화된 연습 문장 제공.
- **최고의 사용자 경험**: 원고지 UI를 통해 정확한 띄어쓰기를 익히고, 아빠의 응원 메시지로 동기 부여.
- **편리한 설치(PWA)**: 별도의 앱스토어 설치 없이 홈 화면에 추가하여 앱처럼 사용.

## 3. 주요 기능 (Special Features for Yul)

### 📱 3.1. PWA (Progressive Web App) 기능
- **홈 화면 추가**: 아이콘 디자인부터 율이가 좋아할 만한 스타일로 설정.
- **오프라인 모드**: 인터넷이 불안정한 곳에서도 학습 가능하도록 캐싱 처리.

### 🖋️ 3.2. 원고지 감성 UI
- 율이의 눈이 편안한 아이보리 톤의 원고지 배경.
- 글자를 클릭하여 오답을 체크할 때마다 부드러운 애니메이션 효과.

### 🔊 3.3. 아빠의 응원과 가이드 (TTS+)
- **맞춤형 TTS**: 속도 조절은 물론, 오답 시 "괜찮아, 다시 해보자!" 같은 따뜻한 피드백.
- **칭찬 시스템**: 정답을 맞히면 화면 가득 하트나 폭죽 애니메이션을 보여줌.

### 🔍 3.4. 스마트 오답 노트 & 스티커 판
- **틀린 글자 집중 케어**: 자주 틀리는 단어는 '복습 단어'로 지정하여 반복 노출.
- **나만의 스티커 판**: 학습을 완료할 때마다 디지털 스티커를 부여하여 성취감 고취.

## 4. 기술 스택 (Simple & Powerful)
- **Frontend**: Next.js (App Router)
- **Styling**: Vanilla CSS (따뜻한 파스텔 톤 테마)
- **PWA**: `next-pwa`를 활용하여 매니페스트 및 서비스 워커 설정
- **Storage**: 브라우저 로컬 스토리지 (율이의 학습 기록을 안전하게 기기에 저장)

## 5. 단계별 개발 로드맵
1.  **Step 1: PWA 기초 설정** 및 따뜻한 느낌의 UI 테마 구축.
2.  **Step 2: 원고지 채점창 컴포넌트** 및 TTS 기능 구현.
3.  **Step 3: 오답 체크 로직** 및 동기 부여 애니메이션 추가.
4.  **Step 4: 오답 관리** 및 스티커 판 기능 구현.
5.  **Step 5: 실제 기기 테스트** 및 율이에게 선물하기! 🎁
