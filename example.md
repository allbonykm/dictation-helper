# 📝 율이 받아쓰기 문제 관리 가이드

아빠가 율이를 위해 문제를 추가하거나 바꾸고 싶을 때 아래의 방법을 따라해 보세요.

## 📍 문제 데이터의 위치
문제는 현재 다음 파일에 모여 있습니다:
`src/app/practice/page.tsx`

---

## 🛠️ 문제 추가/수정 방법

### 1. 파일 열기
`src/app/practice/page.tsx` 파일을 메모장이나 코드 에디터로 엽니다.

### 2. 문장 리스트 찾기
파일 상단에 있는 `SAMPLE_SENTENCES`라는 부분을 찾습니다.

```tsx
const SAMPLE_SENTENCES = [
  "나비가 꽃밭에 앉아 있어요.",
  "학교 가는 길에 친구를 만났어요.",
  "우리 아빠는 요리를 정말 잘해요."
];
```

### 3. 원하는 문장으로 바꾸기
따옴표(`" "`) 사이에 율이가 연습할 문장을 적고, 쉼표(`,`)로 구분해 주면 됩니다.

**예시 (1학년 받아쓰기 1급):**
```tsx
const SAMPLE_SENTENCES = [
  "우리 가족은 행복해요.",
  "나무가 쑥쑥 자라요.",
  "맛있는 사과를 먹어요.",
  "선생님 안녕하세요.",
  "친구랑 즐겁게 놀아요."
];
```

---

## 💡 팁
- **문장 부호**: 마침표(`.`), 물음표(`?`), 느낌표(`!`) 등도 정확하게 넣어주시면 율이가 나중에 부호까지 공부하기 좋습니다.
- **길이**: 원고지 칸은 문장 길이에 맞춰 자동으로 늘어납니다. 너무 긴 문장보다는 율이가 공책 한 줄에 적기 적당한 길이가 좋습니다.
- **반영 확인**: 파일을 저장하면 실행 중인 앱에 즉시 적용됩니다!
